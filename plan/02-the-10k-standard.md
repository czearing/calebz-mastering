# The 10k Standard

Why this reads as a flagship build, not a template. Premium is emergent. No single effect carries it. Hierarchy, type, color, motion, copy, and speed must resonate as one.

## The five tells of a cheap site (banned)

1. Generic hero with a stock gradient and a centered headline.
2. Bar-style audio visualizers and waveform confetti.
3. Section reveals that all fade-up the same way at the same speed.
4. Stacked marketing adjectives with no proof.
5. Jank on scroll. Layout shift. Slow first paint.

## The craft details that buy premium

Award-level sites obsess over micro-details on top of a clean, usable frame. Usability is roughly 30 percent of an Awwwards score, so craft never costs speed.

| Detail | Standard |
|--------|----------|
| Easing | One custom cubic-bezier used everywhere. No default ease. |
| Hover | Every interactive element has a considered hover and focus state. |
| Type | One typeface, tuned tracking, optical sizes, no orphan lines. |
| Rhythm | A single spacing scale. Nothing off-grid. |
| Motion | Slow, intentional, staggered. Reads as control, not decoration. |
| Sound | One restrained WebGL motif, not a literal spectrum. |
| Copy | Fewer words than expected. Every line earns its place. |

## The one signature

Two layers. The interaction signature is the A/B before and after as the hero: the visitor hears a raw mix become a master in one tap (see 23-signature). The visual signature is a single slow neon-cyan field reacting to scroll velocity, pointer, and the playhead. It is the only heavy visual and never reacts to live audio amplitude. Restraint is the flex.

## The feel in three words

Quiet. Exact. Fast.

## The test

Open it on a mid-tier phone. If first paint is instant, scroll is smooth, and the one 3D element still feels intentional, it reads as a 10k build. If any of those slip, cut the effect before shipping.
